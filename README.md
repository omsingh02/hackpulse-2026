## What this is ?

A polished, mobile-first registration and info page for a student hackathon. Includes hero, event details, tracks, timeline, FAQ, organizer section, and registration CTA.

## Sections

1. Hero with event badge, date, and location
2. What participants get — highlight cards
3. Tracks — themed problem statement areas
4. Event timeline
5. FAQ
6. Organizer section
7. Registration CTA
8. Footer

## Tech stack

- HTML
- CSS (custom properties, mobile-first, cyan accent theme)
- No frameworks, no build tools

## Run locally

```bash
python3 -m http.server 8080
# Open http://localhost:8080
```
